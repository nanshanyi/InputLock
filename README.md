# InputLock

macOS 输入法锁定工具，防止输入法被意外切换。

## 功能

- 锁定当前输入法，切换应用时自动恢复
- 状态栏显示当前输入法图标，锁定时右下角显示锁标记
- 菜单栏快速切换和锁定目标输入法
- 支持快捷键锁定/解锁

## 安装

1. Clone 仓库
2. Xcode 打开 `InputLock.xcodeproj`
3. Cmd+R 运行

## 要求

- macOS 13.0+
- Xcode 15+
