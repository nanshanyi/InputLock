## 功能

- 锁定当前输入法，切换应用时自动恢复
- 状态栏显示当前输入法图标，锁定时右下角显示锁标记
- 菜单栏快速切换和锁定目标输入法
- 支持快捷键锁定/解锁

## 安装

### 方法一：下载 DMG（推荐）

直接下载 [InputLock 1.0.dmg](https://github.com/nanshanyi/InputLock/releases/download/v1.0.0/InputLock%201.0.dmg)，拖入应用程序文件夹即可。

### 方法二：源码编译

1. Clone 仓库
2. Xcode 打开 `InputLock.xcodeproj`
3. Cmd+R 运行
